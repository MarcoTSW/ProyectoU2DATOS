# ProyectoU2DATOS
Proyecto final - examen 2 - estructura de datos 


## Descripción
Implementación de un grafo con lista de adyacencia y aplicación de un algoritmo greedy para la coloración de vértices.

## Funcionalidades
- Creación de grafos no dirigidos
- Coloreo greedy
- Orden natural y por grado descendente
- Validación de coloración

## Pruebas
- Grafo completo K4
- Grafo ciclo C5
- Comparación de órdenes

## Tecnologías
- Python
- Google Colab
